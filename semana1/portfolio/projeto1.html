Processo Seletivo Future4


E como esse projeto foi feito?


O desafio proposto pela equipe da Future4 foi justamente montar um site baseado somente em uma foto enviada para os participantes. Os participantes por sua vez tinham que utilizar tanto HTML, quanto CSS e Javascript para alcançar o resultado desejado nesse projeto. Incluindo, porém não limitado a certos guidelines como a utilização da função "Alarm" com ajuda do javascript.
